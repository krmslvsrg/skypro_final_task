Git

  • git init - Создание репозитория
  • git status - Команда показывает изменения, которые были внесены и готовы для коммита.
  • git commit -m "комментарий к изменению" - Команда сохраняет измения и позволяет добавить короткий комментарий.
  • git add - Команда подготавливает файлы к коммиту
  • git push - Команда используется для отправки коммитов из локального репозитория в удаленный репозиторий.
